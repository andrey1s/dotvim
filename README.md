# dotvim

## Installation

```bash
cd ~
git clone --recursive https://github.com/andrey1s/dotvim.git .vim
ln -s .vim/.vimrc
```

## Hotkeys

`\e` - NERDTree  
`\b` - Buffer list   
`<Shift-Tab>` - Next buffer
