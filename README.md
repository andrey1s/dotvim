#dotvim

## Installation

```bash
cd ~
git clone --recursive https://github.com/andrey1s/dotvim.git .vim
ln -s .vim/.vimrc
```

##Bundles

- [NERDTree][1]
- [bufexplorer][2]
- [syntastic][3]
- [vim-go][4]
- [supertab][5]
- [fugitive][6]

##Hotkeys

`<F4>` - NERDTree  
`<F5>` - Buffer list   
`<F6>` - Prew buffer   
`<F7>` - Next buffer   
`<F12>` - View file list
